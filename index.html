客户端共有两个任务队列：

一个是逻辑任务队列，处理业务逻辑任务，另一个是 http 任务队列，处理 http 请求。

任务被抽象为一个对象，取名为 Operation。这是一个抽象类，所以的任务都需要从这个类派生，重写行为函数 process。这么做的好处就是降低了系统间的耦合度，而且新的任务很容易添加到系统中去。
